# Margin Bottom

## Definition and Usage

The  `marginBottom` property sets the bottom margin of an element.

:::caution Note

Negative values are not permitted.

:::
 
![Example banner](./img/MarginBottom.png)
 
## Syntax
```js"
marginBottom : number
```
